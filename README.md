# countpp

`countpp` is an advanced, multi-device event counting and timestamping platform.

This repository is organized to support:
- Sensor-first recording (accelerometer, audio, camera, touch).
- Installable and no-install recording clients.
- Multi-device session synchronization via a backend server.
- Fast real-time and post-hoc analysis pipelines.
- Decoupled UI components that consume stable APIs.
